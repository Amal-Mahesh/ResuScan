## 📖 Overview

Recruiters spend considerable time manually reviewing resumes to identify suitable candidates. **ResuScan** automates this process by leveraging Natural Language Processing (NLP) to analyze resumes, extract relevant skills, evaluate ATS compatibility, and recommend matching job roles.

The system helps streamline recruitment by reducing manual effort while providing candidates with valuable feedback about their resumes.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Amal-Mahesh/ResuScan.git
```

Navigate into the project directory

```bash
cd ResuScan
```

Install the required packages

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app_v2.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```
